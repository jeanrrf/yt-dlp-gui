## Project Overview
This project is a desktop application built with **Tauri** and **Vue 3** (TypeScript) for managing video downloads using `yt-dlp`.

## Getting Started
- `pnpm install`
- `pnpm dev` — frontend only (Vite)
- `pnpm tauri dev` — full app

## Structure
See `src/` and `src-tauri/` for frontend and backend sources respectively.
